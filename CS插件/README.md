# CS插件



## PushPlus2

修改自项目[CS-PushPlus](https://github.com/lintstar/CS-PushPlus)，在原来上线自动提醒的基础上增加了如下功能：

* 上线自动截图

* 上线自动进程迁移
* 上线执行自定义shell命令
* 上线键盘记录 (新增2023-09-11)
